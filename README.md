# General

now you can know when tnt explode

# Feature
- Can Custom Text
- Can set Make Explosion Break Block Or Not
- Can Set TNT Explode Time
- Can set on placed spawn
- Can set TNT Make Damage or Not
- Can Set TNT Make Knockback or Not
